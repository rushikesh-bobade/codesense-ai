# CodeSense AI 🔍

An AI-powered code review platform that catches security flaws, performance issues, and subtle bugs before they reach production.

## What You're Getting

A complete toolkit for intelligent, automated code review.

- 🔍 **Deep Code Analysis** — AI-powered reviews that go beyond linting. Catches security vulnerabilities, performance bottlenecks, and logic errors.

- ⚡ **Lightning-Fast Performance** — Pages load instantly, interactions are smooth, and your app works great even on slower connections.

- 🎯 **Consistent Design System** — Every color, spacing, and animation follows a cohesive design language. Clean, professional, and polished.

- 📱 **Works Everywhere** — Desktop, tablet, mobile — adapts beautifully to any screen size.

- 🤖 **Powered by Llama 3.3 70B** — Running on Groq infrastructure for ultra-fast AI inference.

## Tech Stack

| Technology    | Purpose                     |
|---------------|-----------------------------|
| React Router  | Full-stack React framework  |
| TypeScript    | Type-safe development       |
| Vite          | Build tooling               |
| Groq + Llama  | AI code analysis engine     |
| Radix UI      | Accessible UI primitives    |
| Tabler Icons  | Icon system                 |
